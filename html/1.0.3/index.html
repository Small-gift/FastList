help_outline 使用说明
  • 选择需要加速的平台（Github或SourceForge）
  • 粘贴需要下载的原始链接到输入框
  • 对于Github，可以点击"测试节点延迟"自动选择最优节点
  • 节点颜色表示延迟状态：绿色(快速) < 100ms，黄色(中等) 100-300ms，红色(慢速) > 300ms
  • 对于SourceForge，可以从下拉菜单中选择加速节点
  • 点击"生成加速链接"获取加速后的下载链接
  • 点击"一键下载"直接开始下载
